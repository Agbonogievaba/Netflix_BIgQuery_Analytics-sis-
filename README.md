# Netflix Content Analysis (BigQuery & SQL)

## Project Overview
An analysis of Netflix's library growth from 2015-2021, focusing on the rise of international content versus US-based titles.

## Key Insights
* International Surge:*International content peaked around 2019-2020.
* India's Growth: India showed a massive spike in content additions in 2018.
* Content Mix: The United States remains the largest contributor, with a heavy leaning toward Movies over TV Shows.
